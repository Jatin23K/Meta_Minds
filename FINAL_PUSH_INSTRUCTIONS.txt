================================================================================
                        TROUBLESHOOTING
================================================================================

Error: "fatal: not a git repository"
Solution: Make sure you ran "git init" first

Error: "failed to push some refs"
Solution: Use --force flag: git push origin main --force

Error: "permission denied"
Solution: Check Git credentials or use personal access token

Error: "venv still showing up"
Solution: Run "git rm -r --cached venv" then commit and push again
